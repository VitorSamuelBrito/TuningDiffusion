# TuningDiffusion

[![Library functions test](https://github.com/VitorSamuelBrito/TuningDiffusion/actions/workflows/ci-test.yml/badge.svg)](https://github.com/VitorSamuelBrito/TuningDiffusion/actions/workflows/ci-test.yml)

The tuning diffusion method generates perfectly diffusive one-dimensional Q(t) trajectories from a given diffusion coefficient dependent on the coordinate D(Q) and the free energy profile F(Q).
